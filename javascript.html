Positivos Negativos
JavaScript é empregado em todos os lugares da web Pode ser difícil desenvolver aplicativos grandes, embora você também use a sobreposição de TypeScript
Ele oferece o poder de criar interfaces ricas Se o erro ocorrer no JavaScript, ele pode parar para renderizar todo o site. Os navegadores são extremamente tolerantes com erros de JavaScript
Existem muitos projetos de código aberto que fornecem uma ajuda útil na adição de JavaScript do desenvolvedor Não importa a proporção que o JavaScript interpreta rapidamente, o JavaScript DOM (Document Object Model) é lento e pode ser uma renderização nunca rápida com HTML
